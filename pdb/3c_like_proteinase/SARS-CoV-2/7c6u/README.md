# PDB 7c6u

## Overview

**Protein name**: 3c like proteinase

**Organism**: SARS-CoV-2

**Method**: X-Ray Diffraction

## Description

Crystal structure of the SARS-CoV-2 main protease in complex with GC376, forming a homodimer.

## Basefolder

7c6u.pdb and 7c6u.cif - the coordinates of the threedimensional molecular model

7c6u-sf.cif - deposited diffraction data for this PDB entry
