# PDB 1ssk

## Overview

**Protein name**: nucleocapsid protein

**Organism**: SARS-CoV

**Method**: NMR

## Description

This NMR structure shows the N-terminal RNA-binding domain of SARS-CoV nucleocapsid protein (N). It is a monomer with a length of 158 amino acids and exhibits a five-stranded Î²-sheet. This structure binds single-stranded RNA to enable the packaging of the viral genome RNA into a helical ribonucleocapsid (RNP).
