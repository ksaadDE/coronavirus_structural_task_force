# PDB 7kj5

## Overview

**Protein name**: surface glycoprotein

**Organism**: SARS-CoV-2

## Description

EM structure of the SARS-CoV-2 spike glycoprotein prefusion with one RBD in up conformation. Homo trimer with three chains.
