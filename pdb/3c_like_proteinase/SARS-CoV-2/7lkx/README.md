# PDB 7lkx

## Overview

**Protein name**: 3c like proteinase

**Organism**: SARS-CoV-2

**Method**: X-Ray Diffraction

## Description

Crystal structure of 3C-like protease from SARS CoV2 (Homodimer) in complex with inhibitor 3e. The resolution is 1.6 Å.

## Basefolder

7lkx.pdb and 7lkx.cif - the coordinates of the threedimensional molecular model

7lkx-sf.cif - deposited diffraction data for this PDB entry

7lkx.mtz - structure factors after automatic refinement of molecular model against the diffraction data.
