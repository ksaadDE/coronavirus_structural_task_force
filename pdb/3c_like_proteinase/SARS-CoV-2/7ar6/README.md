# PDB 7ar6

## Overview

**Protein name**: 3c like proteinase

**Organism**: SARS-CoV-2

**Method**: X-Ray Diffraction

## Description

Full length 3C-like protease from SARS-CoV-2. It has a large beta angle and the spacegroup is C2. This structure was resolved using x-ray crystallography to identify possible binding compounds in approved drugs or drugs in clinical trial. Resolution is 1.40A.

## Basefolder

7ar6.pdb and 7ar6.cif - the coordinates of the threedimensional molecular model

7ar6-sf.cif - deposited diffraction data for this PDB entry

7ar6.mtz - structure factors after automatic refinement of molecular model against the diffraction data.
