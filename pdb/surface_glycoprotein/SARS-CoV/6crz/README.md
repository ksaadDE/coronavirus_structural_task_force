# PDB 6crz

## Overview

**Protein name**: surface glycoprotein

**Organism**: SARS-CoV

**Method**: Cryo-EM

## Description

SARS Spike Glycoprotein, Trypsin-cleaved, Stabilized variant, C3 symmetry. This study reveal that stabilized spikes resist conformational changes induced by receptor.
