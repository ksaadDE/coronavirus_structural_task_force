# PDB 7jyc

## Overview

**Protein name**: 3c like proteinase

**Organism**: SARS-CoV-2

**Method**: X-Ray Diffraction

## Description

Crystal structure of 3C-like protease from SARS CoV2 (Homodimer) in complex with covalent inhibitor narlaprevir. Narlaprevir is an hepatitis C serine protease inhibitor.

## Basefolder

7jyc.pdb and 7jyc.cif - the coordinates of the threedimensional molecular model

7jyc-sf.cif - deposited diffraction data for this PDB entry

7jyc.mtz - structure factors after automatic refinement of molecular model against the diffraction data.
