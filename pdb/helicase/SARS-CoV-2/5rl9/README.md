# PDB 5rl9

## Overview

**Protein name**: helicase

**Organism**: SARS-CoV-2

**Method**: X-Ray Diffraction

## Description

Crystal Structure of the SARS-CoV-2 helicase in complex with Z1703168683. Its ligands are zinc ions, phosphate ions and 1-(3-fluoro-4-methylphenyl)methanesulfonamide.
