# PDB 7ax6

## Overview

**Protein name**: 3c like proteinase

**Organism**: SARS-CoV-2

**Method**: X-Ray Diffraction

## Description

Crystal structure of 3C-like protease from SARS CoV2 (Homodimer) bound to Glutathione isopropyl ester. This compound binds covalently to Cys156.

## Basefolder

7ax6.pdb and 7ax6.cif - the coordinates of the threedimensional molecular model

7ax6-sf.cif - deposited diffraction data for this PDB entry

7ax6.mtz - structure factors after automatic refinement of molecular model against the diffraction data.
