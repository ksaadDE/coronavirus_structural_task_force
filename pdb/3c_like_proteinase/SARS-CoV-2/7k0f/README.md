# PDB 7k0f

## Overview

**Protein name**: 3c like proteinase

**Organism**: SARS-CoV-2

**Method**: X-Ray Diffraction

## Description

Full length 3C-like protease from Sars-Cov-2 in complex with a deuterated GC376 alpha-ketoamide analog. The resolution is 1.65A and it was solved with X-ray diffraction. [VR4](https://www.rcsb.org/ligand/VR4) is bound 

## Basefolder

7k0f.pdb and 7k0f.cif - the coordinates of the threedimensional molecular model

7k0f-sf.cif - deposited diffraction data for this PDB entry

7k0f.mtz - structure factors after automatic refinement of molecular model against the diffraction data.
