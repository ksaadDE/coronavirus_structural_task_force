## Description

Part of PanDDA analysis group deposition. SARS-CoV-2 Nsp3 macrodomain crystallographic fragment screen. The potential binding ligand(s): [W3Y](https://www.rcsb.org/ligand/W3Y).

5s4i.pdb and 5s4i.cif - the coordinates of the threedimensional molecular model

5s4i-sf.cif - deposited diffraction data for this PDB entry
