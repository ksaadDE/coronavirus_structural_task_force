## Description

Part of PanDDA analysis group deposition. SARS-CoV-2 Nsp3 macrodomain crystallographic fragment screen with EU-OPENSCREEN fragment library. The potential binding ligand(s): [WS4](https://www.rcsb.org/ligand/WS4).
