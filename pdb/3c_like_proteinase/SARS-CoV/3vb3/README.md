# PDB 3vb3

## Overview

**Protein name**: 3c like proteinase

**Organism**: SARS-CoV

**Method**: X-Ray Diffraction

## Description

Full length 3C-like protease from SARS-CoV forming a dimer. This X-ray crystal structure was phased using PDB entry 2q6g for molecular replacement & has data to a resolution of 2.2Å.

## Basefolder

3vb3.pdb and 3vb3.cif - the coordinates of the threedimensional molecular model

3vb3-sf.cif - deposited diffraction data for this PDB entry

3vb3.mtz - structure factors after automatic refinement of molecular model against the diffraction data.
