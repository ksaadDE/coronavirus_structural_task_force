# PDB 7aku

## Overview

**Protein name**: 3c like proteinase

**Organism**: SARS-CoV-2

**Method**: X-Ray Diffraction

## Description

Crystal structure of 3C-like protease from SARS CoV2 (Homodimer) bound to Calpeptin. It binds covalently via its aldehyde group to Cys145, forming a thiohemiacetal. This peptidomimetic inhibitor occupies substrate pockets S1 to S3. 

## Basefolder

7aku.pdb and 7aku.cif - the coordinates of the threedimensional molecular model

7aku-sf.cif - deposited diffraction data for this PDB entry

7aku.mtz - structure factors after automatic refinement of molecular model against the diffraction data.
