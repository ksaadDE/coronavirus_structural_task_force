# PDB 6waq

## Overview

**Protein name**: surface glycoprotein

**Organism**: SARS-CoV

**Method**: X-Ray Diffraction

## Description

Crystal structure of the SARS-CoV-1 RBD bound by the cross-reactive single-domain antibody SARS VHH-72.

## Basefolder

6waq.pdb and 6waq.cif - the coordinates of the threedimensional molecular model

6waq-sf.cif - deposited diffraction data for this PDB entry

6waq.mtz - structure factors after automatic refinement of molecular model against the diffraction data.
