# PDB 7kmk

## Overview

**Protein name**: surface glycoprotein

**Organism**: SARS-CoV-2

**Method**: Cryo-EM

## Description

This electron microscopy structure shows the SARS-CoV-2 spike protein in complex with Fab 15033-7 bound to two RBDs.
