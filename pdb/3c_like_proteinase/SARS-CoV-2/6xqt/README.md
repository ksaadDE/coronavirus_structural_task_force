# PDB 6xqt

## Overview

**Protein name**: 3c like proteinase

**Organism**: SARS-CoV-2

**Method**: X-Ray Diffraction

## Description

This X-ray crystal structure shows the SARS-CoV-2 main protease in complex with Narlaprevir. The malleability of the SARS-CoV-2 3CL M pro active-site cavity facilitates the binding of clinical antivirals.

## Basefolder

6xqt.pdb and 6xqt.cif - the coordinates of the threedimensional molecular model

6xqt-sf.cif - deposited diffraction data for this PDB entry

6xqt.mtz - structure factors after automatic refinement of molecular model against the diffraction data.
