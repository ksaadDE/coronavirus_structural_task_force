# PDB 6xbh

## Overview

**Protein name**: 3c like proteinase

**Organism**: SARS-CoV-2

**Method**: X-Ray Diffraction

## Description

Full length 3C-like protease from SARS-CoV with the peptide like enzyme inhibitor UAW247 in the active site. This X-ray crystal structure was phased using PDB entry 6yb7 for molecular replacement & has data to a resolution of 1.6Å.

## Basefolder

6xbh.pdb and 6xbh.cif - the coordinates of the threedimensional molecular model

6xbh-sf.cif - deposited diffraction data for this PDB entry

6xbh.mtz - structure factors after automatic refinement of molecular model against the diffraction data.
