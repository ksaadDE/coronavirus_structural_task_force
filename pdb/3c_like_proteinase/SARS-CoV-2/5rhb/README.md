# PDB 5rhb

## Overview

**Protein name**: 3c like proteinase

**Organism**: SARS-CoV-2

**Method**: X-Ray Diffraction

## Description

Full length 3C-like protease from SARS-CoV-2 forming a dimer. This protein is bound to a small molecule frag, (E)-1-(pyrimidin-2-yl)methanimine. This X-ray crystal structure has data to a resolution of 1.58Å

## Basefolder

5rhb.pdb and 5rhb.cif - the coordinates of the threedimensional molecular model

5rhb-sf.cif - deposited diffraction data for this PDB entry

5rhb.mtz - structure factors after automatic refinement of molecular model against the diffraction data.
