# PDB 5rfj

## Overview

**Protein name**: 3c like proteinase

**Organism**: SARS-CoV-2

**Method**: X-Ray Diffraction

## Description

Full length 3C-like protease from SARS-CoV-2 forming a dimer. This protein is bound to a small molecule frag, N-(4-methoxy-1,3-benzothiazol-2-yl)acetamide. This X-ray crystal structure has data to a resolution of 1.80Å

## Basefolder

5rfj.pdb and 5rfj.cif - the coordinates of the threedimensional molecular model

5rfj-sf.cif - deposited diffraction data for this PDB entry

5rfj.mtz - structure factors after automatic refinement of molecular model against the diffraction data
