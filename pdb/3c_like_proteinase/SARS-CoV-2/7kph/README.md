# PDB 7kph

## Overview

**Protein name**: 3c like proteinase

**Organism**: SARS-CoV-2

**Method**: X-Ray Diffraction

## Description

1.45 Å X-ray diffraction crystal structure of SARS-CoV-2 main protease, which is responsible for cutting the polyprotein into functional parts, in mature form.

## Basefolder

7kph.pdb and 7kph.cif - the coordinates of the threedimensional molecular model

7kph-sf.cif - deposited diffraction data for this PDB entry

7kph.mtz - structure factors after automatic refinement of molecular model against the diffraction data.
