# PDB 7lkw

## Overview

**Protein name**: 3c like proteinase

**Organism**: SARS-CoV-2

**Method**: X-Ray Diffraction

## Description

Crystal structure of 3C-like protease from SARS CoV2 (Homodimer) in complex with inhibitor 3d (deuterated analog of inhibitor 3c). The resolution is 1.65 Å.

## Basefolder

7lkw.pdb and 7lkw.cif - the coordinates of the threedimensional molecular model

7lkw-sf.cif - deposited diffraction data for this PDB entry

7lkw.mtz - structure factors after automatic refinement of molecular model against the diffraction data.
