# PDB 7cbt

## Overview

**Protein name**: 3c like proteinase

**Organism**: SARS-CoV-2

**Method**: X-Ray Diffraction

## Description

Crystal structure of 3C-like protease from SARS CoV2 (Homodimer) in complex with protease inhibitor GC376.

## Basefolder

7cbt.pdb and 7cbt.cif - the coordinates of the threedimensional molecular model

7cbt-sf.cif - deposited diffraction data for this PDB entry

7cbt.mtz - structure factors after automatic refinement of molecular model against the diffraction data.
