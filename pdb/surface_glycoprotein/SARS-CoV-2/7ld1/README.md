# PDB 7ld1

## Overview

**Protein name**: surface glycoprotein

**Organism**: SARS-CoV-2

**Method**: Cryo-EM

## Description

This electron microscopy structure shows the SARS-CoV-2 Spike protein in complex with Receptor Binding Domain antibody DH1047.
