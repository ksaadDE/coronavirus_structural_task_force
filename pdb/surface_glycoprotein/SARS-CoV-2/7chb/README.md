# PDB 7chb

## Overview

**Protein name**: surface glycoprotein

**Organism**: SARS-CoV-2

**Method**: X-Ray Diffraction

## Description

Crystal structure of SARS-CoV-2 RBD in complex with the antibody BD-236. Non polymer only trimer, spike with one chain, antibody with one heavy and one light chain. Antibody binds to the RBD, one related structure 7che, it shows the antibody together with the anitbody BD-368-2. 

## Basefolder

7chb.pdb and 7chb.cif - the coordinates of the threedimensional molecular model

7chb-sf.cif - deposited diffraction data for this PDB entry

7chb.mtz - structure factors after automatic refinement of molecular model against the diffraction data.
