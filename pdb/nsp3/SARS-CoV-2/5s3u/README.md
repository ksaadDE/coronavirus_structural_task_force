## Overview

**Protein name**: nsp3

**Organism**: SARS-CoV-2

## Description

Part of PanDDA analysis group deposition. SARS-CoV-2 Nsp3 macrodomain crystallographic fragment screen. The potential binding ligand(s): [W2J](https://www.rcsb.org/ligand/W2J).

5s3u.pdb and 5s3u.cif - the coordinates of the threedimensional molecular model

5s3u-sf.cif - deposited diffraction data for this PDB entry
