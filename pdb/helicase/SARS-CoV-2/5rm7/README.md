# PDB 5rm7

## Overview

**Protein name**: helicase

**Organism**: SARS-CoV-2

**Method**: X-Ray Diffraction

## Description

Crystal Structure of the SARS-CoV-2 helicase in complex with Z69118333. Its ligands are zinc ions, phosphate ions and ~{N}-(4-hydroxyphenyl)-3-phenyl-propanamide.
