# PDB 6m2q

## Overview

**Protein name**: 3c like proteinase

**Organism**: SARS-CoV-2

**Method**: X-Ray Diffraction

## Description

Full length 3C-like protease from SARS-CoV. This X-ray crystal structure was phased using PDB entry 6lu7 for molecular replacement & has data to a resolution of 1.7Å.

## Basefolder

6m2q.pdb and 6m2q.cif - the coordinates of the threedimensional molecular model

6m2q-sf.cif - deposited diffraction data for this PDB entry

6m2q.mtz - structure factors after automatic refinement of molecular model against the diffraction data.
