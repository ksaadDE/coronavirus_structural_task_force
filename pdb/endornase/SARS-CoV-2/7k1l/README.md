# PDB 7k1l

## Overview

**Protein name**: endornase

**Organism**: SARS-CoV-2

**Method**: X-Ray Diffraction

## Description

The crystal structure of the nsp15 endoribonuclease in complex with uridine-2',3'-vanadate. Further ligands are sulfate ion, acetate ion and 1,2-ethanediol.

## Basefolder

7k1l.pdb and 7k1l.cif - the coordinates of the threedimensional molecular model

7k1l-sf.cif - deposited diffraction data for this PDB entry

7k1l.mtz - structure factors after automatic refinement of molecular model against the diffraction data.
