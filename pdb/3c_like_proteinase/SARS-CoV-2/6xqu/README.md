# PDB 6xqu

## Overview

**Protein name**: 3c like proteinase

**Organism**: SARS-CoV-2

**Method**: X-Ray Diffraction

## Description

Room-temperature X-ray diffraction data. Structure of the main-protease of SARS-CoV-2, in complex with Boceprevir.

## Basefolder

6xqu.pdb and 6xqu.cif - the coordinates of the threedimensional molecular model

6xqu-sf.cif - deposited diffraction data for this PDB entry

6xqu.mtz - structure factors after automatic refinement of molecular model against the diffraction data.
