# PDB 7jr4

## Overview

**Protein name**: 3c like proteinase

**Organism**: SARS-CoV-2

**Method**: X-Ray Diffraction

## Description

Crystal structure of 3C-like protease from SARS CoV2 (Homodimer) with alternative conformation of the active site promoted by methylene-bridged cysteine and lysine residues. The resolution is 1.55 Å.

## Basefolder

7jr4.pdb and 7jr4.cif - the coordinates of the threedimensional molecular model

7jr4-sf.cif - deposited diffraction data for this PDB entry

7jr4.mtz - structure factors after automatic refinement of molecular model against the diffraction data.
