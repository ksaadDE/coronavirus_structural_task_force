# PDB 6ynq

## Overview

**Protein name**: 3c like proteinase

**Organism**: SARS-CoV-2

**Method**: X-Ray Diffraction

## Description

Full length 3C-like protease from SARS-CoV with a 2-Methyl-1-tetralone ligand bound. This X-ray crystal structure was phased using PDB entry 6yb7 for molecular replacement & has data to a resolution of 1.8Å.

## Basefolder

6ynq.pdb and 6ynq.cif - the coordinates of the threedimensional molecular model

6ynq-sf.cif - deposited diffraction data for this PDB entry

6ynq.mtz - structure factors after automatic refinement of molecular model against the diffraction data.
