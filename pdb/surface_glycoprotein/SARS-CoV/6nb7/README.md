# PDB 6nb7

## Overview

**Protein name**: surface glycoprotein

**Organism**: SARS-CoV

**Method**: Cryo-EM

## Description

SARS-CoV complex with human neutralizing S230 antibody Fab fragment (state 2). This study reveal that stabilized spikes resist conformational changes induced by receptor.
