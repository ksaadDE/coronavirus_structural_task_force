# PDB 5rl0

## Overview

**Protein name**: 3c like proteinase

**Organism**: SARS-CoV-2

**Method**: X-Ray Diffraction

## Description

Part of PanDDA analysis group deposition SARS-CoV-2 main protease deposition of computational designs of SARS-CoV-2 main protease covalent inhibitors. SARS-CoV-2 main protease from Coronaviridae sp. screened against computationally designed covalent inhibitors by X-ray Crystallography at the XChem facility of Diamond Light Source beamline I04-1. The potential binding ligand(s): [VEG](https://www.rcsb.org/ligand/VEG).

5rl0-sf.cif - deposited diffraction data for this PDB entry
